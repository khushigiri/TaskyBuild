
# Tasky Build - Task Management Web App

Tasky Build is a responsive task management web application that allows users to create, edit, search, and delete tasks dynamically. It uses JavaScript and `localStorage` to persist data, ensuring tasks remain available even after refreshing the page.

---

## Live Demo

https://khushigiri-tasky-build.netlify.app/

---

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome

---

## Key Features

- Dynamic task card generation using JavaScript template literals
- Persistent data storage using browser `localStorage`
- In-place task editing with `contenteditable`
- Real-time task search functionality
- Responsive UI using Bootstrap grid system

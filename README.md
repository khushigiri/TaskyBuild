
# Tasky Build - Full Stack Logic Task Manager

Tasky Build is a dynamic, client-side task management application. It bridges the gap between a static UI and a functional app by utilizing JavaScript's `localStorage` for data persistence and `contenteditable` attributes for seamless in-place editing.

## Tech Stack

- HTML5
- Bootstrap 5
- CSS3
- FontAwesome
- JavaScript

---


## Key Features

* Dynamic UI Rendering: Automatically generates task cards using JS Template Literals.

* Persistent Storage: Data is saved locally in the browser; your tasks remain even after a page refresh.

* In-Place Editing: Update task titles, types, and descriptions directly on the card without opening a separate form.

* Mobile-First Design: Features a dedicated bottom-docked button for mobile users and a responsive Bootstrap grid for desktops.

* Search Engine: A built-in filtering system that scans your task list in real-time.
